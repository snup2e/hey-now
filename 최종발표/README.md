# 최종발표 덱 (Hey now!)

임베디드시스템설계 최종발표용 **단일 파일 HTML 슬라이드 덱**. 브라우저에서 `index.html`
더블클릭으로 실행(오프라인 자체완결, Pretendard 폰트 임베드). 발표는 **녹화한 시연영상 + 이 덱**으로 진행.

## 실행 / 조작
- 열기: `index.html` 더블클릭.
- 이동: `→ ← ↓ ↑ Space` / 스크롤, `Home`/`End`, `F` 전체화면, `?` 단축키, 목차 클릭 점프.
- 캡처 모드: `index.html?shot=N` → N번 슬라이드만 애니메이션 없이 정적 표시(스샷/PDF 변환용).
- 2D 덱 구조: **섹션 = 세로(row), 섹션 내 여러 슬라이드 = 가로(track).** 우측에 섹션 내비.

## 배포 산출물 (발표 현장에 가져갈 것)
- **HTML 덱**: `최종발표/` **폴더째** 복사(상대경로 자산 7개: woff2·demo.mp4·figures 5장). 브라우저로 오프라인 실행. 참조 자산 누락 없음 확인됨.
- **PDF (PPT 대용)**: `Hey_now_최종발표.pdf` — 12페이지(슬라이드 1장=1쪽), 2560×1440. 생성법 = `?shot=0..11` 헤드리스 크롬 PNG(scale 2) → Pillow로 결합(`Image.save(..., save_all=True)`). **데모 페이지(10)는 PDF라 영상 대신 "하차벨이 울립니다" 포스터** → 발표 시 그 대목에서 `demo.mp4` 따로 재생. 덱 수정 후 재생성하면 됨.
- **발표 대본**: `발표대본.md` — 7분(약 6분25초+영상), 슬라이드별, 2인 분할 예시([A]1~7 / [B]8~12). KWS·CMN 등 용어는 말로는 쉬운 표현으로(슬라이드엔 유지). ★8·9가 핵심 의사결정(역이름 분류 접고 출발 신호음 카운트로 전환).

## 문구 정책 (중요)  ← 사용자 지시
- **비전문가가 모를 만한 "Claude식/번역투" 용어만** 쉬운 말로. 바꾼 6개:
  디바운스 파이프라인→"연속 검출로 오인식 차단" · 실차→실제 열차 · cross-trip→처음 타는 열차 ·
  결정론적 재생→"같은 녹음 그대로 재생" · 트립→탑승/회 · 차임→**출발 신호음**.
- **기본 기술용어는 그대로 유지**: KWS, CNN, log-mel, CMN, INT8, ProtoNet, GRL, X-CUBE-AI,
  recall, LOO, 도메인 갭, activation 등. (한 번 과하게 풀었다가 원상복귀시킨 이력 있음 — 다시 풀지 말 것)
- **성능 수치는 단일 accuracy/held-in val 대신 confusion matrix(TP/FN/FP)로.** 검출(KWS·차임)은
  이벤트 단위라 TN이 정의 안 됨(연속 시간축) → 열 제목 "검출함/검출 안 함", TN 칸은 **"정상 기각(TN)"**
  라벨(숫자 X). 다클래스 분류(13-class 등)는 13×13 불가라 정확도 유지. 메모리 `feedback-present-with-confusion-matrix`.
